## Design Notes

### AI Document Parsing

The `event_imports` and `event_reports` enhancements support AI-powered document parsing:

1. **Upload**: Document uploaded, stored in blob storage
2. **Parse**: AI extracts data, stores in `extracted_data` JSONB
3. **Confidence**: AI confidence score tracked
4. **Review**: Manual review workflow with notes
5. **Generate**: Creates structured `event_reports` from parsed data

### Payroll System

New `roles` and `EmployeeDeduction` tables support payroll:

- **Roles**: Define job roles with base pay rates and overtime rules
- **Deductions**: Track employee-specific deductions (health insurance, 401k, garnishments)
- **Calculation**: Application calculates net pay using role rates and deductions

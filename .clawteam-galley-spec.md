## What Galley Does That We Need to Match

### Recipe Management
- Rich recipe editor with inline ingredient editing (qty, unit, name in a table row)
- Yield/scale control prominently placed (top-right), instant scaling of all ingredients
- Sub-recipe support (recipe-within-recipe for sauces, stocks, etc.)
- Cost per yield and cost per container visible in recipe header
- Food cost % calculation
- Procedure steps with duration, temperature, equipment tags, tips, images
- Version history with diff comparison
- Recipe sharing & favorites

### Menu Management
- Course-based menu builder with drag-and-drop dish ordering
- Menu cycle planning (weekly/monthly rotation)
- Budget-aware menu building (set target cost, see live margin as you add dishes)
- Allergen matrix view across entire menu
- Dietary breakdown summary per menu
- Menu templates (save & reuse configurations)
- Production plan generation from menus

### Costing & Analysis
- Real-time ingredient cost from vendor catalog integration
- Recipe cost rollup (ingredients → recipe → dish → menu)
- Margin analysis dashboard (food cost %, profit per serving, menu-level margins)
- Cost comparison across recipe versions
- Serving-based vs yield-based pricing

### Visual Appeal (Galley's UI Principles)
- Clean, airy layouts with generous whitespace
- Prominent data hierarchy — costs and key metrics are never hidden
- Grouped field sections with clear labels
- Expandable/collapsible sections (description, HACCP, notes)
- Professional card designs with subtle gradients and hover effects
- Responsive layouts that wrap field groups together
- Consistent iconography (chef hat, dollar, clock, users)

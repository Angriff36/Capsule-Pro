## Event Data

Each event contains:

| Field | Description |
|-------|-------------|
| Title | Event name |
| Event Number | Unique identifier |
| Date | Event date and time |
| Status | Current lifecycle state |
| Guest Count | Expected number of guests |
| Location | Venue address |
| Client | Linked CRM client |
| Dishes | Menu items for the event |
| Prep Tasks | Kitchen preparation tasks |
| Source Documents | Uploaded PDFs, CSVs, and images |

## Dashboard

The events dashboard at `/events` shows:

- Total events count
- Upcoming events
- Total expected guests
- Confirmed vs. tentative breakdown
- Quick access to create, import, and view reports

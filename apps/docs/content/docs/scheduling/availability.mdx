---
title: Availability
description: Employee availability windows and weekly patterns.
---

Availability records define when employees are available to work.

## Availability Properties

| Field | Description |
|-------|-------------|
| Employee | Staff member |
| Day of Week | Which day(s) |
| Start/End Time | Available time window |
| Effective Dates | When this availability applies |

## Features

- **Create** individual availability entries
- **Batch create** weekly patterns (e.g., "Available Mon-Fri 8am-4pm")
- **Overlap detection** — System warns if new entries overlap existing ones
- **Filter** by employee, date range
- **Employee view** — See all employees' availability for a date range (scheduling view)

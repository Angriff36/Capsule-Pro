## Dashboard

The scheduling dashboard at `/scheduling` displays real-time data from 10+ SQL queries:

- Total staff count and scheduled hours
- Open shifts needing coverage
- Labor cost summary
- Daily breakdown
- Today's shifts
- "Shift Battle Arena" leaderboard

All metrics update in real-time via Ably WebSocket connection.

---
title: Labor Budgets
description: Budget tracking with threshold alerts at 80%, 90%, and 100%.
---

Labor budgets set spending limits for staff costs with automatic threshold alerts.

## Budget Types

- **Event** — Budget for a specific event
- **Weekly** — Rolling weekly labor budget
- **Monthly** — Monthly labor budget

## Budget Properties

| Field | Description |
|-------|-------------|
| Name | Budget name |
| Type | Event, Weekly, or Monthly |
| Amount | Budget limit |
| Status | Active, Paused, or Archived |
| Utilization | Current spend as percentage of budget |

## Threshold Alerts

The system generates alerts at three thresholds:
- **80%** — Warning: approaching budget limit
- **90%** — Critical: near budget limit
- **100%** — Over budget

Alerts can be acknowledged and resolved.

## Features

- Create, edit, and delete budgets
- Filter by type and status
- Search by name
- View utilization metrics
- Manage alert acknowledgments

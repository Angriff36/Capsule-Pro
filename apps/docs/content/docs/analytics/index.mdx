The Analytics module provides cross-cutting dashboards for operational intelligence.

## Dashboards

<CardGroup cols={2}>

  <Card title="Kitchen Analytics">

  </Card>
  <Card title="Finance">

  </Card>

## API Endpoints

| Endpoint | Description |
|----------|-------------|
| `/api/analytics/kitchen` | Kitchen performance data |
| `/api/analytics/finance` | Financial metrics |
| `/api/analytics/staff/summary` | Staff summary |
| `/api/analytics/staff/employees/:id` | Individual employee analytics |
| `/api/analytics/events/profitability` | Event profitability data |

## Data Sources

Analytics queries run directly against the database using raw SQL for performance. They join across multiple tenant schemas (`tenant_kitchen`, `tenant_events`, `tenant_staff`, `tenant_inventory`) to provide cross-domain insights.

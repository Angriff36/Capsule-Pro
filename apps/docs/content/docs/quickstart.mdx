## Getting Started

### Step 1: Clone Your Repository

Clone the documentation repository from GitHub following the [GitHub cloning instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository).

### Step 2: Preview Your Changes

Install and run the development server:

```bash
npm i -g mintlify
mintlify dev
```

Run this command at the root of your documentation (where `mint.json` is located).

### Step 3: Deploy Your Changes

Commit and push your changes to Git. The changes will be automatically deployed to your documentation site.

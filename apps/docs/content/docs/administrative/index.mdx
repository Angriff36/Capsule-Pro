The Administrative module provides leadership tools for operational oversight.

## Production Meeting Prep

The main administrative page at `/administrative` is designed for weekly production meetings:

- **Week-based navigation** — Browse events by week
- **Event cards** with staff counts and document attachments
- **Week-over-week statistics** for trend comparison
- **Search and filter** across events
- **Source file upload** — Attach documents to events

## Overview Boards

The strategic overview at `/administrative/overview-boards` runs 14 parallel queries to display:

- Event readiness status
- Kitchen throughput metrics
- Scheduling gap analysis
- Budget alerts
- Executive action items
- Board health indicators

This is a read-only leadership dashboard.

## Operational Kanban

The kanban board at `/administrative/kanban` provides a 4-column task board:

| Column | Description |
|--------|-------------|
| Backlog | Tasks not yet started |
| In Progress | Active work |
| Review | Awaiting review |
| Done | Completed |

Tasks can be created and moved between columns.

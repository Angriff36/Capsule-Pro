Capsule Pro is a multi-tenant operations platform built for catering companies. It covers the full lifecycle of catering operations — from client acquisition and event planning through kitchen production, inventory management, staff scheduling, payroll, and analytics.

## Architecture Highlights

- **Multi-tenant** — Composite primary keys with automatic tenant scoping across all data
- **Manifest IR** — Custom DSL that compiles to an Intermediate Representation governing all domain mutations through Policies, Constraints, Guards, Actions, and Events
- **Real-time** — Transactional outbox pattern with Ably for live updates across kitchen, scheduling, and command board
- **Offline-capable** — Mobile views for kitchen tasks, prep lists, and waste logging with sync queues
- **25+ integrations** — Clerk (auth), Stripe (payments), Ably (realtime), Sentry (errors), OpenAI (AI), Liveblocks (collaboration), Resend (email), and more

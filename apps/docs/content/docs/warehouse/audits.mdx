Warehouse audits use cycle count sessions to verify physical inventory against system records.

## Session Lifecycle

| Status | Description |
|--------|-------------|
| Created | Session is set up |
| In Progress | Counting is underway |
| Completed | All counts are recorded |
| Finalized | Variances reconciled, stock adjusted |
| Cancelled | Session was aborted |

## Session Management

From the audits page:
- **Create** a new session
- **Start** / **Continue** counting
- **Complete** when all items are counted
- **Finalize** to reconcile and adjust stock
- **Cancel** to abort

## Filters

Sessions can be filtered by:
- Status
- Type (full count, spot check, etc.)

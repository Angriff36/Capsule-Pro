## Key Capabilities

- **70+ API endpoints** across recipes, dishes, menus, stations, prep lists, tasks, waste, allergens, and inventory
- **Manifest IR commands** for all mutations — every create, update, and state change flows through the Manifest pipeline
- **Real-time production board** with Ably WebSocket updates for task claims and progress
- **Offline mobile support** with sync queues for kitchen tasks, prep lists, and waste entries
- **AI-powered prep task generation** from event data
- **Recipe versioning** with immutable versions and cost tracking

# unresolved/

Canonical area — **open questions**, not features. Holds `uncertainty.md` records filed from [`../_templates/uncertainty.md`](../_templates/uncertainty.md).

Each uncertainty is one file: `unresolved/<area>.<name>.<question>.md`. Cross-link it from the related canonical unit and leave `Ryan final decision` as `NEEDS-RYAN`.

> **Before filing here:** `cd canonical && treex` to see existing uncertainties. Do not grep — a question framed differently won't match your search and you'll file a duplicate.

## The data layer (memorize — answers "what do I call in a page")

Three generated files (regenerated by `pnpm manifest:client`, never hand-edited):

```
manifest-types.generated.ts     entity TS types
manifest-client.generated.ts    async listX()/getX() + executeCommand → Convex (read-bridge, ONE-SHOT)
manifest-hooks.generated.ts     TanStack hooks over the client (~unused — decide: adopt or delete)
```

| Page/component is… | Use | Real-time? |
|---|---|---|
| **Server** (RSC / action), static/detail/report | `await client.listX()` / `getX()` / `executeCommand` | ❌ one-shot |
| **Client + needs LIVE** (board, kanban, dashboard) | `useQuery(api.queries.listX)` from `convex/react` | ✅ live |

The async client uses `client.query()` (one-shot) — it **cannot** be real-time. Real-time exists
**only** via native `convex/react`. Choose per surface; most CRUD is fine one-shot.
